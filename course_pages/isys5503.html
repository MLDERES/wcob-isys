Course Details

ISYS 5103: Data Management

Course Description

This course covers the fundamental concepts of data management and database systems. Students learn how to gather data from a variety of sources, including databases, and focus on retrieving, manipulating, and analyzing data using SQL. The course also covers the basics of programming using Python and how to use Python to manipulate data.

Course Work

Industry Skills Acquired

Completing this course will equip students with fundamental skills in:

  • Problem-solving and analytical thinking
  • Python programming
  • SQL Queries
