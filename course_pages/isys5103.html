Course Details

ISYS 5103: Data Management

Course Description

This foundational course covers the basic concepts of computer science, including algorithms, data structures, and programming paradigms. It's designed to give students a broad overview of the field and its applications.

Course Work

  • Weekly programming assignments
  • Bi-weekly quizzes on computational theory
  • Group project focused on developing a simple application
  • Final exam covering all course materials

Skills Acquired

Completing this course will equip students with fundamental skills in:

  • Problem-solving and analytical thinking
  • Basic principles of algorithm development
  • Understanding of various programming languages
  • Ability to collaborate on software projects
